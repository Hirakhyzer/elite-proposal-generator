# Elite Proposal Generator

A white and gold React app for creating polished client-ready project drafts for Elite Era Development L.L.C.

Brand style: white background, black text, and `#f4af00` gold accents.

## Features

- Clean white and gold UI
- Client information form
- Service selector
- Engagement tiers
- Budget and timeline inputs
- Project context field
- Generated document preview
- Summary section
- Deliverables section
- Project phases section
- Terms and next steps section
- Markdown, JSON, and TXT downloads
- Footer credit: Made by Hira Khyzer

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Vitest

## Run Locally

```bash
npm install
npm run dev
```

## Run Tests

```bash
npm test
```
